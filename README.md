# Credit Card Customer Segmentation
This project applies Machine Learning clustering techniques to segment credit card customers based on their financial behavior and card usage patterns.

The main goal of this project is to discover hidden customer groups using the K-Means clustering algorithm. These customer segments can help businesses improve marketing strategies, customer retention, and financial services.

This is an unsupervised learning project because the dataset does not contain predefined labels or target classes.

## Result
The model grouped customers into 3 clusters:
- Inactive Customers
- VIP Customers
- Cash Advance Users
